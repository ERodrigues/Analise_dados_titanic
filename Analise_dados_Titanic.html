Análise de dados - Titanic¶

História¶

No incio do sculo 20 o Titanic zarpou do porto de Southampton com destino a Nova York com cerca de 892 tripulantes e 2435 passageiros. No trejeto o navio ainda passou pelos portos de Cherbourg-Octeville na Frana e Queenstown na Irlanda.

O navio no chegou ao seu destino em virtude de uma coliso com um iceberg no dia 14 de abril e acabou afundando no dia seguinte com mais de 1500 pessoas a bordo.

Fonte¶

Nesta anlise ser utilizado a fonte de dados train.csv do site Kaggle que contm um conjunto de dados dos passageiros do Titanic.

Métodos¶

Foram criados dois mtodos para a anlise dos dados com as seguintes finalidades:

  1. return_percentual: calular e retornar o percentual de acordo com os valores passados evitando repetio de cdigo
  2. categorie_age_range: mtodo que retornar em qual faixa etria se encontra o passageiro de acordo com sua idade
In [224]:
# Funo utilizada para retornar o percentual de acordo com os parmetros passados.
def return_percentual (value_counts, count):
    return round((value_counts/count)*100)
In [225]:
def categorie_age_range (age):
    if age <= 12:
        return 'Crianca'
    elif age > 12 and age < 61:
        return 'Adulto'
    else:
        return 'Idoso'

Limitações¶

Foram encontrados alguns valores limitantes por no conter os dados para todos os passageiros, como so os casos dos atributos:

  • Cabin
  • Age
  • Embarked

O atributo Cabin foi retirado das anlises no momento de data wrangling. J o atributo Age foi preenchido com a mdia da idade presente no Data Frame. Com o atributo Embarked foi adicionado uma informao de valor no informado para que pudessemos identificar que para alguns passageiros no foi informado onde o mesmo embarcou.
